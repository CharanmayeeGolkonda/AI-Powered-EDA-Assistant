# 📊 AI-Powered EDA Assistant

An AI-powered Exploratory Data Analysis (EDA) web application that helps users analyze datasets, generate visualizations, and obtain AI-generated insights using Google Gemini AI.

---

## 🚀 Features

- 📂 Upload CSV or Excel datasets
- 📊 Automatic Exploratory Data Analysis (EDA)
- 📈 Generate charts and visualizations
- 🤖 AI-powered insights using Google Gemini
- 💬 Chat with your dataset
- 📄 Generate PDF reports
- 🌐 Interactive web interface built with Streamlit

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Gemini API
- ReportLab

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/CharanmayeeGolkonda/AI-Powered-EDA-Assistant.git
```

Move into the project folder:

```bash
cd AI-Powered-EDA-Assistant
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it (Windows):

```bash
venv\Scripts\activate
```

Install the dependencies:

```bash
pip install -r requirements.txt
```

---

## 🔑 Configure API Key

Create a `.env` file and add:

```text
GEMINI_API_KEY=YOUR_API_KEY
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

Open your browser and go to:

```
http://localhost:8501
```

---

## 📈 Workflow

1. Upload a CSV or Excel file.
2. The application performs Exploratory Data Analysis.
3. Charts and statistical summaries are generated.
4. Gemini AI provides insights.
5. Users can ask questions about the uploaded dataset.
6. Reports are generated automatically.
